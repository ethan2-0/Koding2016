More Information About Spaghetti Compute


Spaghetti Compute is a web service that provides cheap computing
power to customers who need to solve problems that cannot be
solved on a single machine. Spaghetti Compute using computing
power from browsers accross the world to solve costumer problems. All the
customer needs to do, it submit his/her problem via a client script and an
HTTP server following our specification.

We handle the rest; we get your code running on browsers, across the world,
cheaper than competing providers such as EC2.

Our goal is simple: to replace advertisements on the web. Join us in making
the web better for the user.
